# Battleship Game

A simple Battleship game built with React (frontend) and Node.js (backend).

## Prerequisites

- Node.js installed (v14+ recommended)

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/aldask/Battleship-game.git
cd Battleship-game
```

### 2. Start the Backend

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   node server.js
   ```
   or if you prefer Nodemon
   ```bash
   nodemon server.js
   ```
   The backend runs on `http://localhost:3001`.

### 3. Start the Frontend

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the React application:
   ```bash
   npm start
   ```
   The frontend runs on `http://localhost:3000`.

---

### Playing the Game

1. Open `http://localhost:3000` in your browser.
2. The game will initialize automatically on page load.
3. Enjoy the Battleship game!
